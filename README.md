# Hello Prisma

This is a learning exercise project using Prisma. It's a simple application that uses Prisma to interact with a database.

## Disclaimer

Please note that this project is just a learning exercise and the goal is to understand how to use Prisma. It's not intended to be used in a production environment.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- You have installed Node.js and npm.
